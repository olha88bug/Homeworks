  1. git status - показує статус в локальному репозитрії
  2. git push - Записує оновлення коду на віддалений репозиторій
  3. git add . - гіт починає відслідковувати зміни що внесені в файл
  4. git commit -m “commit text” - зберігає поточний стан файла
  5. git branch - показує список локальних гілок
  6. git branch name - створює гілку з ім’ям name
  7. git checkout -b name - створює нову гілку з ім’ям “name” і переходить в неї
  8. git checkout name - перехід до потрібної гілки
  9. git push -u origin name - перший пуш нової локальної гілки на гітхаб
