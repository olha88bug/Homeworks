Селектори

Можна розділити CSS селектори на п’ять категорій:
  1. Прості селектори (вибір елементів на основі імені, ідентифікатору, класу). Приклад: .highlight ; id
  2. Комбінаторні селектори (вибір елементів на основі визначених відношень між ними).
    В CSS є чотири різних комбінатора:
    1. селектор нащадків (пробіл)
    2. селектор дочірній (>)
    3. селектор сусідній родинний (+)
    4. селектор загальний родинний (~)
  3. Селектори псевдокласів (вибір елементів на основі визначеного стану)
    4. Псевдо-клас використовується для визначення особливого стану елемента
    • Стилізація елемента, коли користувач наводить на нього курсор
    • Стилізація відвіданих та невідвіданих посилань по-різному
    • Стилізація елемента, коли він знаходиться в фокусі
    Приклад:
    a:visited {
    color: #00FF00
  4. Селектори псевдоелементів (вибір і стилізація частини елемента)
    5. CSS псевдо-елемент використовується для стилізації визначених частин елемента
    • Стилізація першої літери або рядка елемента
    • Вставка змісту до або після змісту елемента
    Приклад:
    Псевдо-елемент ::first-line використовується для додавання спеціального стилю до першого рядка тексту.
  5. Селектори атрибутів (вибір елементів на основі атрибуту або значення атрибуту)
    6. Можна стилізувати елементи HTML, які мають визначені атрибути або значення атрибутів
    Приклад:
    Селектор [attribute] використовується для вибору елементів зі вказаним атрибутом.
